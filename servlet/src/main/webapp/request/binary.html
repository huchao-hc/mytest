method="get" enctype="any one"
method="post" enctype="application/x-www-form-urlencoded"
当 表单的 enctype 属性取值为 "application/x-www-form-urlencoded" ( 也是默认值 ) ，并且采用 post 方式提交表单时，
则 在发送前对所有字符进行编码 ( 将空格转换为 "+" 符号，特殊字符转换为 ASCII HEX 值 )
method="post"
当 表单的 enctype 属性取值为 "text/plain" ，并且采用 post 方式提交表单时，
表单中的所有的控件都以key=value形式传递给服务器，并且以纯文本形式发送(不会编码特殊字符)、每个key=value独占一行

method="post" enctype="multipart/form-data"
当 表单的 enctype 属性取值为 "multipart/form-data" ，并且采用 post 方式提交表单时， 表单中的每个控件都是当前表单对应的消息的一部分
如果需要通过表单提交字节数据给服务器，则必须指定 method 为 "post" 、enctype 为 multipart/form-data
