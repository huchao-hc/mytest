比着 "葫芦" ( prototype ) 画 "瓢" (class)

Object是一个对象(class) ，不是一个实例(instance)

Object对象(class)首先是个函数，是通过 Function 对象(class)来构造的，因此 Object.constructor 返回 Function
任何对象(class)都有自己的原型(prototype)，通过对象(class)的原型(prototype)可以定义一个对象(class)的实例(instance)中所包含的属性和函数
Object对象(class)的实例(instance) 是通过 Object 对象(class)本身的构造函数来构造的，所以 Object.prototype.constructor 返回 Object
